The Zero-Config
NAS Operating System

RoqNAS turns clean Debian servers into high-performance, resilient storage nodes. Complete with interactive ZFS growth wizards, automated Bonjour local discoverability, and native Apple Time Machine sharing.

$ curl -sSL https://update.roqnas.org | bash

Copies installation bootstrapper. Installs core dependencies & launches console control plane daemon.

Premium Architecture

Engineered for Reliability and Simplicity

RoqNAS provides professional-grade NAS capabilities wrapped inside a beautiful, zero-configuration user interface.

ZFS Volume Growth Engine

Manage datasets and extend ZFS pools dynamically using VDev expansion, drive replacements, or direct RAIDZ group attachments.

macOS Apple Time Machine

Samba export layers configured with native `vfs_fruit` extensions, allowing you to deploy secure network backup destinations with custom storage quotas.

Bonjour Discoverability

Zero-configuration LAN multicast DNS (mDNS) broadcasting. Exposes file-shares and administration panels to your network using custom device icons.

KVM Virtual Machine Node

Spin up, modify, and monitor host-level kernel virtual machines (KVM) directly from the unified admin portal.

Advanced Authentication & MFA

Lock down your control portal with Multi-Factor Authentication (MFA/TOTP) and enforce terminal console user re-authentication.

Update Manager

Verify and apply upgrades dynamically. Connects natively to git-based release channels (release, beta, dev branches).

Deploy RoqNAS in 60 Seconds

Set up a minimal installation of Debian 13, and deploy the entire server infrastructure using a single bootstrap script.

root@roqnas-installer:~
# 1. Access terminal of a clean Debian 13 instance
# 2. Run the single-line deployment bootstrapper
curl -sSL https://update.roqnas.org | bash
